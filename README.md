# 可视化新新魔塔存档编辑器  

### 修改楼层地块信息
墙体、怪物、宝物、暗墙……的位置。
### 修改各项数值
等级、攻击、防御、金币……  
道具数量……  
每回合攻击次数、攻击临界值、暴击率……  
### 其他杂项
楼层传送器可传送的楼层范围；  
商店所需的金币；  
勇士攻击动画；  
勇士当前状态（中毒、衰弱、正常）；  
勇士当前章的等级。  

# 快速开始
1、运行flashplayer_sa.exe并打开Nwe Magic Tower.swf；  
2、进入游戏后，在主塔 入口，按下E，点击栏位1的箭头，保存存档，之后关闭游戏；  
3、运行本软件，点击打开存档，不出意外，会自动定位到存档目录，直接双击sol文件打开；  
4、修改各项数据后，点击保存存档即可。  
PS：1.42魔塔存在作弊器，按下E后再按H，可进入作弊器界面。  

# 一些说明
1、经测试三个章可同时生效。可以是直接修改章等级，也可以是清空魔塔5F所有的墙体来获得所有的章，所有的章均可正常鉴定、升级。  
2、在最底层用神剑之证击杀假公主后，地图会全部变为岩浆，可以通过修改器修改出楼梯，可以正常达到其他层。  
3、复活十字架：在战斗中，生命值如果归0，消耗该道具，可以立即恢复2000点生命值，且战斗继续。该道具正常只能获得一个，但是实际修改中，数量可以＞1，也就是一场战斗十字架可以多次生效，取决于你的十字架数量。  
4、章等级数值框中填入的数值需严格遵守右方规则，否则可能会导致修改失效。只要章LV≥1，章就会生效，多个章可同时生效。需要注意的是，在老人处鉴定章时，章变得可用时，会增加部分能力，比如勇者之证就会增加5%暴击率，而在修改器中直接修改章等级，并不会获得这部分属性。
