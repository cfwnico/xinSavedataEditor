# 可视化新新魔塔存档编辑器  

## 修改楼层地块信息
墙体、怪物、宝物、暗墙……的位置。
## 修改各项数值
等级、攻击、防御、金币……  
道具数量……  
每回合攻击次数、攻击临界值、暴击率……  
## 其他杂项
楼层传送器可传送的楼层范围；  
商店所需的金币；  
勇士攻击动画  
勇士当前状态

# 测试环境
系统版本：windows11\10\7  
软件版本：python3.10  
播放器版本：FlashPlayer32  
游戏版本：新新魔塔1.42作弊版  

# 已知问题
1.不支持新新魔塔1.42以外的版本，因为1.1版本似乎是假存档。  
2.依赖.net framework 4.5。

# 提示
软件会查找最新修改的sol文件地址并自动定位。建议先开游戏存个档，方便软件自动定位。
